pagehelper的使用

是 mybatis 进行 分页的插件

核心步骤

  1. 引入pagehelper 依赖
  2. 在application.yml中进行配置
  3. 在调用mapper 层方法前 ， 使用pagehelper.start（pagenum，pagesize），它之后的一个查询会进行分页查询
  4. 返回结果 使用PageInfo对象包装

pageInfo对象

所以要获取查到的列表 直接getlist 即可
要获取其他参数也是直接get

注意

startpage 只对它之后的一个查询起作用
所以避免 startpage之后有其他查询意外触发了这个

使用步骤

  1. 引入依赖 pagehelper 与mybatis ，web ， mysql

  1. 写 application.yml

  2. 定义model

  3. 定义mapper

  4. 定义service
    在调用mapper曾方法前 使用pagehelper.start（pagenum，pagesize）
    要返回分页对象使用PageInfo对象进行包装

  5. 定义controller 自测
