编程社区项目

首先这个项目的目的

  • 使用练习学习的技术栈（为了技术而技术）
    • 目前已有的技术：
      • rabbitmq
      • nginx（部署用）
      • docker
      • es
      • springsecurity
      • redis等
  • 我的设想是 一个oj （可以用上docker，rabbitmq）+博客社区（es，redis等）的网站
  • 我将这个项目命名为：cheems 编程炫技社区

项目开发的流程

需求分析=>设计（概要设计、详细设计）=>技术选型=>初始化／引l入需要的技术=>写Demo=>写代码（实现业务逻辑）=>测试（单元测试、系统测试）=>代码提交／代码评审=>部署=>发布上线
