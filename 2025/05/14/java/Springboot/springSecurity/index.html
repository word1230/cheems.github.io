spring Security

springSecurity

springsecurity 可以对用户进行身份认真，还可以进行权限管理等操作

一、springSecurity入门

基于aop思想和Servlet过滤器实现的框架
提供了 认证与授权功能 ， 实现用户只能访问自己有权限的资源

实现权限管理通常设计三个对象

  • 用户：包括用户名，密码，其他用户信息， 用于认证功能
  • 角色：主要包含 角色名称，角色描述，权限信息 。 用于授权操作
  • 权限：也叫菜单 包含 权限名称，url地址等 。 用于动态展示 ，只显示用户有权限的部分
