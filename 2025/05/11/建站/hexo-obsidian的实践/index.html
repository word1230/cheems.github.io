hexo+obsidian的实践

简介

  1. 如何使用obsidian来写hexo博客，集成terminal来发布

  2. obsidian+picgo +cdn +github 图床 来保存图片
    图片保存在本地太占用空间，而且博客想要发布在如csdn/知乎等，又需要上传图片，很麻烦，所以使用图床来保存图片而不是在本地保存

  3. 使用quickadd +js脚本 发布 + 创建博客

集成terminal

  1. 安装插件terminal
  2. 修改快捷键

将 ctrl +t 快捷键 原本的功能删掉
改为打开根目录外部终端

这样使用 ctrl +t 就可以打开终端
进行hexo 命令

picgo

  1. 安装
    进入安装界面 https://github.com/Molunerfinn/PicGo/releases 找到2.3.1版本
    选择windows 的 exe 文件

按照步骤进行安装

  1. 配置

打开picgo设置
我们只使用github做图床， 其他的不显示 ，其它的勾掉

图床设置 -> github 填入参数

例如
仓库名：就是 https://github.com/abcde/picture 中 github.com 后面的
分支名： main
存储路径： 会有一个文件夹来存放， 比如 会将图片存储在 /resource目录下
自定义域名： 使用cdn 进行加速
格式： https://cdn.jsdelivr.net/gh/ + 仓库名

生成token

在github 点击头像 –> setting -> developer setting -> personal acess tokens ->tokens(classic)
-> generate new token

点击generate token ，即可获取token ，复制 并粘贴到picgo中

  1. 在 obsidian 插件市场中下载image auto upload

就可以实现粘贴图片到obsidian，就会上传
